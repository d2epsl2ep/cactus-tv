# Changelog

## v0.6.0

- Jellyfin / Emby 连接管理移入 `/admin.html#media`，前台只保留用户切换。
- 媒体服务器管理页增加 Jellyfin、Emby 连接教程。
- 支持同一服务器保存多个用户，单用户与聚合模式切换。
- 媒体库直接读取服务器收藏、播放列表、下一集和最近加入。
- 增加服务器原生推荐与基于收藏、近期观看类型的本地排序。
- 增加电视遥控器方向键、确认键、返回键和播放暂停键适配。
- 新增播放列表独立路由 `/playlist/:provider/:id`。
- 不新增 Cloudflare 绑定、数据库表或环境变量。


## v0.5.0

### Jellyfin / Emby

- 可以在前台设置中连接多个 Jellyfin 或 Emby 服务器。
- 支持账号密码登录，也支持访问令牌与用户 ID。
- 媒体连接保存在当前浏览器，不新增 D1、KV、R2 或环境变量。
- 新增媒体库页面，显示继续观看、最近加入和服务器媒体库。
- 全站搜索会同时检索已连接的媒体服务器和原有数据源。
- 支持电影、剧集、季、分集、海报、简介、演员和评分。
- 支持原画直连与 HLS 转码备用线路。
- 支持服务器字幕、自动下一集和切换线路保留进度。
- 播放开始、播放中、暂停与停止状态会同步回媒体服务器。
- 媒体播放地址使用临时会话，不把服务器令牌直接放进视频 URL。
- 临时会话失效时，前端会使用本机保存的令牌重新连接一次。

### 文档

- README 改为面向部署者和使用者的说明。
- 增加 Jellyfin、Emby 连接教程和公网 HTTPS 要求。
- 区分“仅使用私人媒体库”和“启用兼容接口后台”两种部署方式。

## v0.4.0

### 播放与容灾

- 播放失败后自动尝试同一数据源的其他线路。
- 同源线路全部失败后，按接口延迟和本机历史成功率尝试其他数据源。
- 自动换源时保留当前播放进度、集数和字幕入口。
- 播放器增加上一集、下一集和手动切换备用线路。
- 播放结束后显示 5 秒倒计时并自动进入下一集，可在设置中关闭。

### 页面与性能

- 增加首页分类入口和搜索筛选。
- 搜索结果分批渲染。
- 增加详情、播放、分类、搜索、收藏和历史直达路由。
- 搜索 API 使用 Cloudflare Cache API，并限制上游请求并发。
